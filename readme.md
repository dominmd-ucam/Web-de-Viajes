# 🌍 WebDeViajes

## 📝 Descripción
WebDeViajes es una plataforma web dedicada a compartir experiencias de viaje, destinos turísticos y consejos para viajeros. Nuestro objetivo es crear una comunidad donde los amantes de los viajes puedan encontrar inspiración y recursos para sus próximas aventuras.

## ✨ Características
- 🔍 Búsqueda de destinos turísticos
- 📸 Galería de fotos de viajes
- 📝 Blog con experiencias de viajeros
- 🗺️ Mapas interactivos de destinos
- 💬 Sección de comentarios y reseñas
- 📱 Diseño responsive para todos los dispositivos

## 🛠️ Tecnologías Utilizadas
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Node.js
- Express
- MongoDB

## 🚀 Instalación
1. Clona este repositorio:
```bash
git clone https://github.com/tuusuario/WebDeViajes.git
```
2. Navega al directorio del proyecto:
```bash
cd WebDeViajes
```
3. Instala las dependencias:
```bash
npm install
```
4. Inicia el servidor:
```bash
npm start
```

## 📋 Requisitos
- Node.js (v14.0.0 o superior)
- MongoDB (v4.0 o superior)
- Navegador web moderno
